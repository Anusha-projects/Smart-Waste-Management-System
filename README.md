# Smart Waste Management System

## Overview
A full-stack waste management platform where users can:
- Report garbage overflow issues
- Earn reward points
- Track complaint status
- Admin can verify reports and manage rewards

## Tech Stack
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features
- User Authentication
- Garbage Reporting
- Reward System
- Admin Dashboard
- Complaint Tracking
